# E-Book capture application

## Features

- OS: Windows, Linux
- Framework: PyQT, OpenCV
- Language: Python

## Quick start

### Docker

#### Run on Dockerfile

```shell script

docker build -t ebook_capture_pyqt .

docker run --name ebook_capture_pyqt -d ebook_capture_pyqt
```
